# coffee2U
Website about coffee &lt;3 

## Requirements
* Yarn ^1.22.0
* Node ^14.16.0

## Instalation
```
git clone https://github.com/najwer23/coffee2U.git
```
## Run in folder client/
```
yarn install
yarn start
```
## Deploy - Run in folder client/
```
yarn run deploy
```
