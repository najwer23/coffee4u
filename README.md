# coffee4u
Website about coffee &lt;3 

## Requirements
* Yarn ^1.22.0
* Node ^14.16.0

## Instalation
```
git clone https://github.com/najwer23/coffee4u.git
```
## Run
```
yarn install
yarn start
yarn build
```
